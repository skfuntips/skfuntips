
SkFunTips
----------------------------------
what is SkFunTips?

SkFunTips is a text post-process tool ,
which makes your text attractive and fun.
just like this:

                  @     @ @ @           @   @       @       @                   
                  @     @   @           @           @       @                   
          @ @     @     @     @         @           @     @ @                   
          @   @ @     @ @   @ @ @     @ @ @           @ @   @ @
